# Lộ trình học Docker theo từng buổi

Bộ lesson này tách từ tài liệu chính để bạn học theo nhịp ngắn, mỗi buổi từ 45 đến 90 phút.

## Danh sách buổi học

1. [Buổi 1 - Tổng quan và cài đặt](./buoi-01-tong-quan-va-cai-dat.md)
2. [Buổi 2 - Image, container và lệnh cơ bản](./buoi-02-image-container-va-lenh-co-ban.md)
3. [Buổi 3 - Dockerfile và build image](./buoi-03-dockerfile-va-build-image.md)
4. [Buổi 4 - Docker Compose cơ bản](./buoi-04-docker-compose-co-ban.md)
5. [Buổi 5 - Thực hành Redis và Seq](./buoi-05-thuc-hanh-redis-va-seq.md)
6. [Buổi 6 - Volumes, networks, biến môi trường và debug](./buoi-06-volumes-networks-env-va-debug.md)

## Cách học đề xuất

- Đọc tài liệu chính để nắm bức tranh tổng thể.
- Học từng buổi theo thứ tự ở trên.
- Sau mỗi buổi, chạy ngay file trong thư mục `practice`.
- Nếu bị lỗi, ưu tiên đọc log trước rồi mới sửa.

## Bộ file thực hành đi kèm

- `practice/nginx-static`: ví dụ Dockerfile cho web tĩnh.
- `practice/redis`: ví dụ Compose cho Redis.
- `practice/seq`: ví dụ Compose cho Seq và seq-input-gelf.
